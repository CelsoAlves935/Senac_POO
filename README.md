# ☕ Programação Orientada a Objetos (Java) - Senac

Este repositório contém meus estudos práticos e exercícios desenvolvidos durante a disciplina de **POO** no curso de **Análise e Desenvolvimento de Sistemas**.

## 🚀 Tecnologias e Conceitos
* **Linguagem:** Java.
* **Conceitos Praticados:** Classes, Objetos, Encapsulamento, Herança e Polimorfismo.
* **Banco de Dados:** Integração básica com MySQL.

## 📂 Estrutura
Os exercícios estão organizados por tópicos de aula para facilitar a consulta da evolução da lógica e sintaxe da linguagem.
